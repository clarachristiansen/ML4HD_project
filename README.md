# ML4HD_project
Final project in the Machine Learning for Human Data course. 

## How to run 
1. Clone the repo
2. Download the dataset. [Available here.]([https://link-url-here.org](https://www.kaggle.com/datasets/yashdogra/speech-commands))
3. Make environment. Specified in requirements.txt but may be old packages and Linux heavy to make it compatible with the DTU HPC.
4. Optional (but can't guarantee everything work without it): Define wandb configs in .env. See relevant fields in wandb_template.env.
5. Use CLI commands to start training, evaluation etc.
